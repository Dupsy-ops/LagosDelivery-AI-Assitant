# LagosDeliver Invoice Formatting Rules

## General Rules
- Company name always appears at the top in capitals: LAGOSDELIVER LIMITED
- All monetary values must show the Naira symbol: ₦
- Use commas for thousands: ₦1,500 not ₦1500
- Dates always in Nigerian format: DD/MM/YYYY
- Invoice numbers follow this format: INV-LD-XXXXX-YYYY
  (XXXXX = tracking number digits, YYYY = current year)

## Layout Rules
- Always use a table for itemized charges
- Total must be clearly separated from itemized list
- VAT line must appear before the final total
- Payment instructions must always appear at the bottom

## Charge Calculation Rules
1. Start with base delivery fee
2. Add weight surcharge if package above 5kg
3. Add interstate surcharge if pickup and delivery are different states
4. Sum all charges = Subtotal
5. Calculate VAT: Subtotal × 7.5% = VAT Amount
6. Final Total = Subtotal + VAT Amount

## Table Format
All charge tables must look like this:
| Description | Amount |
|---|---|
| Base Delivery Fee | ₦0,000 |
| Weight Surcharge | ₦0,000 |
| Interstate Surcharge | ₦0,000 |
| **Subtotal** | **₦0,000** |
| VAT (7.5%) | ₦0,000 |
| **TOTAL** | **₦0,000** |

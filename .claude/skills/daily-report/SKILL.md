## Generating a Daily Delivery Report

When this skill is activated:

1. Ask for the report date if not provided
   - Format: DD/MM/YYYY (example: 01/05/2026)

2. Ask for the following information if not provided:
   - Total deliveries attempted today
   - Total deliveries successful
   - Total deliveries failed
   - Top 3 delivery zones (e.g. Lagos Island, Surulere, Lekki)
   - Total revenue generated in Naira (₦)

3. Generate the report in this exact format:

---

# LagosDeliver Daily Report
**Date:** DD/MM/YYYY
**Generated By:** AI Assistant
**Time:** HH:MM

---

## 📦 Delivery Summary
| Metric | Count |
|---|---|
| Total Attempted | 0 |
| Successfully Delivered | 0 |
| Failed Deliveries | 0 |
| Success Rate | 0% |

## 📍 Top Delivery Zones
1. Zone Name — X deliveries
2. Zone Name — X deliveries
3. Zone Name — X deliveries

## 💰 Revenue Summary
- Total Revenue: ₦0,000
- Standard Deliveries: ₦0,000
- Express Deliveries: ₦0,000

## ⚠️ Issues Flagged
- List any failed deliveries or complaints here
- Note any zones with repeated problems

## ✅ Recommendations
- List 2-3 actionable recommendations for tomorrow

---
*Report generated by LagosDeliver AI Assistant*

---

4. Save the report as a .md file named: report-DD-MM-YYYY.md
5. Confirm to the user that the report has been saved
